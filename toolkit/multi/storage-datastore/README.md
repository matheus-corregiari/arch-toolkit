# Storage DataStore

Persistent `StorageProvider` backed by AndroidX DataStore Preferences.

- Artifact: `io.github.matheus-corregiari:storage-datastore`
- Operational targets: Android, JVM, iOS arm64, iOS simulator arm64
- Stub targets: JS, WasmJS

```kotlin
implementation("io.github.matheus-corregiari:storage-datastore:<latest-version>")
```
