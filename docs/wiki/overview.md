## Objective

Summarise the scope of the toolkit, outline the main modules, and highlight how Kotlin Multiplatform (KMP) shapes the architecture choices.

## Installation

All modules are distributed through Maven Central. See the [root README](../../README.md#-installation-gradle-kotlin-dsl) for version catalog samples and choose the artifacts that match your platform and stability requirements.

## Usage Examples

* Browse each module README for runnable snippets.
* Check the `sample/` directory for integrated demonstrations when available.
* Look at CI workflows under `.github/workflows/` for automation blueprints.
